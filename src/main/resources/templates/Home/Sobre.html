Descrição do Projeto

    Principais Mudanças Durante o Desenvolvimento

    • 1: Substituição do Banco de Dados: Inicialmente, utilizei o banco de dados H2, mas encontrei dificuldades com a necessidade constante de popular os dados para testes. Para melhorar a eficiência, migrei para o PostgreSQL. As informações do H2 ainda estão presentes no projeto e podem ser usadas para testes, se necessário.
      2: Alteração na Estrutura das Listas: Modifiquei a estrutura das listas de carros e clientes, substituindo as tags < tr> por < div>. Essa alteração foi necessária para garantir o funcionamento correto do modo dinâmico.
      3: Personalização das Tabelas: Durante o desenvolvimento, personalizei várias tabelas que utilizei com o Bootstrap para adequá-las ao funcionamento específico do estacionamento. Apesar das personalizações, a essência do Bootstrap foi preservada, demonstrando ser uma ferramenta valiosa para a estruturação do projeto.
    • Spring Boot: Implementação do projeto seguindo o padrão de arquitetura Model-View-Controller (MVC), garantindo uma separação clara de responsabilidades, facilitando a manutenção e a escalabilidade do código.
    • Thymeleaf: Usado como motor de templates para a renderização das views, permitindo a integração eficiente entre o back-end Java e o front-end HTML.
    • Bootstrap: Framework CSS utilizado para o design responsivo e moderno da interface, assegurando compatibilidade com diversos dispositivos e tamanhos de tela.
    • CSS Personalizado: Adicionamos estilizações específicas para atender às necessidades do projeto, proporcionando uma experiência de usuário única e alinhada à identidade visual da empresa.
    • Banco de Dados PostgreSQL: Utilizado como o banco de dados principal do projeto, proporcionando robustez e escalabilidade para o armazenamento e gerenciamento dos dados. O PostgreSQL é conhecido por sua confiabilidade e suporte a recursos avançados de SQL.
